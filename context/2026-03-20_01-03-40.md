# Project Context - 2026-03-20 01:03:40

## Project Overview

Agentic RAG Masterclass - A RAG application with chat and document ingestion interfaces.

## Tech Stack

- **Frontend**: React + Vite + Tailwind + shadcn/ui
- **Backend**: Python + FastAPI
- **Database**: Supabase (Postgres, pgvector, Auth, Storage, Realtime)
- **LLM**: MiniMax (OpenAI-compatible Chat Completions API)
- **Observability**: LangSmith
