# Module 6: Hybrid Search & Reranking

## Context

Module 6 adds hybrid search (BM25 keyword + vector) and reranking to improve retrieval quality. Currently the system only uses vector similarity search which struggles with exact matches, typos, and technical terms. Adding BM25 + RRF + reranking will provide better retrieval for diverse query types.

## Complexity: ⚠️ Medium

Multiple components interact in sequence; BM25 via Postgres full-text search is straightforward, but reranking requires LLM integration.

---
